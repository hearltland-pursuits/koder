## Complete Curriculum

This documentation covers **95 comprehensive CSS topics** organized into 13 categories:

### 1. Fundamentals (7 topics)
Core concepts every CSS developer must know - syntax, selectors, cascade, inheritance.

### 2. Colors (4 topics)
Understanding CSS color systems - RGB, HEX, HSL, and color theory.

### 3. Box Model (9 topics)
The foundation of CSS layout - margin, border, padding, sizing, and units.

### 4. Typography (6 topics)
Styling text content - fonts, Google Fonts, text properties, icons.

### 5. Layout Basics (7 topics)
Fundamental layout techniques - display, position, float, z-index.

### 6. Flexbox (4 topics)
Modern one-dimensional layout system for flexible, responsive designs.

### 7. CSS Grid (8 topics)
Modern two-dimensional layout system for complex grid-based designs.

### 8. Selectors Advanced (6 topics)
Advanced targeting - combinators, pseudo-classes, pseudo-elements, specificity.

### 9. Visual Effects (8 topics)
Animations, transitions, transforms, filters, gradients, and shadows.

### 10. Images & Media (6 topics)
Styling images, galleries, sprites, masking, and responsive media.

### 11. Components (7 topics)
Reusable UI patterns - tables, forms, navigation, buttons, tooltips.

### 12. Responsive Design (7 topics)
Creating websites that work on all devices and screen sizes.

### 13. Modern CSS (6 topics)
Cutting-edge features - CSS variables, functions, at-rules, optimization.
